HTML5

  • HTML5是什么？

    HTML5 是 HTML标准的最新演进版本,它是一个新的 HTML 语言版本包含了新的元素，属性和行为，同时包含了一系列可以被用来让 Web 站点和应用更加多样化，功能更强大的技术。 这套技术往往被称作 HTML5 和它的朋友们，通常简称为 HTML5

h5新特性

  • 新增标签

    其中经常使用的form标签input[type='number']、 input[type='color']、input[type='range']、article、 section、footer、header、nav等标签，极大地提高了页面的排版速度

  • h5混合开发app

    在实际开发中，我们往往需要h5提供一些页面，如app下载页面，可以转发和分享的非app的外部链接， 现在还可以通过h5开发页面最后打包成app,比如流行的react-native、uni-app、iCloud
